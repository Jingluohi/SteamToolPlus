## 一、软件简介

**Steam Tool Plus** 是一款纯本地运行的 Steam 游戏管理工具，完全绿色便携，无需安装，所有数据保存在程序运行目录与 `%appdata%/SteamToolPlus` 配置目录下。

### 主要功能

- **游戏浏览**：以 Steam 风格网格浏览游戏，支持搜索与分页
- **游戏本体下载**：通过 manifest 清单文件直连 Steam 官方服务器下载正版分流文件，支持断点续传与完整性验证
- **免 Steam 补丁注入**：为游戏注入模拟 Steam 运行环境，无需 Steam 客户端即可启动游戏
- **清单入库**：将清单文件导入 Steam，使 Steam 库中显示对应游戏
- **游戏库管理**：统一管理本地游戏，支持添加、删除、启动、统计游玩时长
- **完整配置管理器**：统一配置 DLC、成就、统计、物品、模组、控制器、用户、主配置、Overlay、ColdClient、Lobby、应用分组、音视频、Depots/Branches、DLC 网络、HTTP 物品等所有补丁参数
- **假入库工具**：在 Steam 库中模拟显示已拥有游戏，无需真实清单文件
- **主题与背景**：支持深色/浅色主题与自定义背景图片
- **多语言支持**：内置简体中文与 English

### 系统要求

- Windows 10/11 64 位系统
- 无需 Steam 客户端
- 无需联网（除下载游戏与封面外）

## 三、详细使用步骤

### 3.1 首次使用

1. **下载程序**：从网盘下载最新版本 `SteamToolPlus.exe`
2. **解压文件**：将程序解压到任意目录（建议纯英文路径）
3. **运行程序**：双击 `SteamToolPlus.exe`
4. **首次配置**：程序首次启动时会显示配置引导弹窗，帮助您快速完成基础设置
5. **配置 Steam 路径**（可选）：进入【设置】→【全局设置】配置 Steam 安装目录

#### 首次使用配置弹窗

程序首次启动时会自动显示配置引导弹窗：

- **配置 Steam 路径**：设置 Steam 安装目录，用于清单入库功能
- **配置下载路径**：设置游戏默认下载路径（优先使用 D 盘）
- **配置背景图片**：选择个性化背景主题
- **跳过配置**：可选择跳过，稍后手动在设置中配置

#### 二维码弹窗功能

在游戏详情页点击网盘按钮时，会显示二维码弹窗，方便使用手机扫码访问：
- 支持百度网盘、迅雷网盘等链接
- 手机扫码即可快速访问下载页面
- 点击弹窗外区域或关闭按钮可关闭弹窗

### 3.2 浏览和下载游戏

#### 浏览游戏

1. 点击顶部菜单 **游戏** → **浏览** 进入浏览页面
2. 页面以网格形式展示所有可下载的游戏，每页 25 款（5 列 × 5 行）
3. 使用搜索框按中文名、英文名或游戏 ID 搜索
4. 点击游戏卡片进入详情页

> 浏览页游戏封面图片来自程序根目录 `resources/pic/Game_Cover/` 目录，图片尺寸为 460×215。

#### 游戏详情页下载

1. 进入游戏详情页，切换到 **下载** 标签
2. 在“清单文件选择”区域选择来源类型：
   - **7z 文件**：直接选择清单压缩包
   - **文件夹**：选择包含 `.vdf` / `.lua` 和 `.manifest` 的文件夹
3. 点击“选择 7z 文件”或“选择文件夹”
4. 在“游戏下载路径”处点击 **浏览**，选择游戏下载目录
5. 点击 **开始下载**

> **说明**：方法一【开始下载】通过清单文件直连 Steam 官方服务器下载正版分流文件，下载完成后需要 **注入补丁** 才能游玩。

#### 通过工具下载

1. 点击顶部菜单 **工具** → **本体下载**
2. 切换到“单游戏下载”模式
3. 点击 **选择清单文件夹** 按钮
4. 选择包含 `.vdf` / `.lua` 和 `.manifest` 文件的文件夹
5. 程序自动验证并提取游戏信息（名称、App ID、清单数量）
6. 选择或确认下载路径
7. 点击 **开始下载**
8. 查看下载进度和日志输出

#### 批量下载

适用于同时下载多个游戏：

1. 切换到“批量下载”模式
2. 选择包含多个游戏清单文件夹的父文件夹
3. 程序自动扫描并列出所有可下载的游戏
4. 勾选需要下载的游戏
5. 设置统一的下载路径
6. 点击 **批量下载** 开始下载
7. 可选择下载完成后自动关机

**下载控制与日志管理：**

- **日志目录**：程序在 `%appdata%/SteamToolPlus/log/{game_id}/` 目录下创建游戏专属文件夹
- **断点续传**：若下载进程异常退出，程序会自动检测未完成的 depot，重新启动续传
- **完整性验证**：游戏下载完成后，仍可通过详情页的 **验证完整性** 按钮重新校验文件
- **暂停下载**：点击暂停按钮可停止下载进程
- **下载完成**：自动清理 100% 开头的 JSON 进度文件

### 3.3 游戏库管理

#### 进入游戏库

1. 点击顶部菜单 **游戏** → **库** 进入游戏库页面
2. 左侧显示已下载或导入的游戏列表
3. 右侧显示选中游戏的详细信息

#### 添加游戏到库

- 下载游戏后自动添加到库
- 手动导入：点击 **导入游戏** 按钮，选择游戏目录和主程序 exe，填写游戏信息后导入

#### 启动游戏

1. 在游戏卡片上点击 **开始游戏** 按钮
2. 程序自动记录启动时间和游玩时长
3. 游戏运行时按钮变为 **关闭游戏**，可结束游戏进程

#### 管理游戏

- **编辑游戏**：点击编辑按钮，可修改游戏目录、exe 路径、存档路径、封面
- **卸载游戏**：右键游戏卡片 → 卸载，删除游戏文件但保留游戏数据
- **浏览本地文件**：右键游戏卡片 → 浏览本地文件
- **收藏游戏**：右键游戏卡片 → 收藏/取消收藏

### 3.4 免 Steam 补丁注入

此功能可为游戏注入模拟 Steam 运行环境，无需 Steam 客户端即可启动游戏。

#### 进入补丁页面

点击顶部菜单 **工具** → **免Steam补丁**。

> **状态记忆**：补丁页面会记住您填写的游戏路径、AppID、模拟器模式、DLL 版本、是否使用实验版等状态。只有关闭程序或点击 **重置** 按钮，状态才会恢复初始。

#### 基础配置

**步骤一：选择游戏文件夹**

1. 点击 **选择游戏文件夹** 按钮
2. 浏览并选择游戏安装目录（应包含 `steam_api.dll` 或 `steam_api64.dll`）
3. 程序自动检测文件夹中的 DLL 文件

**步骤二：选择模拟器模式**

| 模式 | 说明 |
|------|------|
| **标准模式** | 替换 `steam_api.dll`，适用于大多数 Steam 游戏 |
| **高级模式** | 替换 `steamclient.dll` + `steam_api.dll`，适用于特殊保护游戏 |

**步骤三：选择 DLL 版本（标准模式下）**

| 版本 | 说明 |
|------|------|
| **稳定版** | 推荐大多数用户使用，兼容性好，稳定性高 |
| **功能版（实验版）** | 包含 Overlay 等高级功能，部分游戏可能有兼容性问题 |

> **注意**：只有在标准模式下勾选了“使用实验版（功能版）”，完整配置管理器中的 **Overlay 配置** 入口才会启用。

**步骤四：配置 Steam App ID**

1. 在 **Steam App ID** 输入框中填写游戏的 App ID
2. 点击 **SteamDB** 链接可查询游戏 App ID
3. 程序会自动生成 `steam_interfaces.txt` 文件

**步骤五：应用配置**

1. 点击 **应用配置** 按钮
2. 程序自动创建 `steam_settings` 文件夹并写入默认配置
3. 自动复制补丁 DLL 到游戏目录
4. 自动备份原文件（添加 `.bak` 后缀）

> **默认值说明**：
> - **存档路径**默认使用 `%appdata%/GSE Saves`
> - **DLC** 默认解锁全部 DLC，不会变为手动指定
> - 所有可选项均不会自动填充示例值

#### 完整配置管理器

点击 **完整配置管理器** 按钮，可统一配置所有补丁参数。

**配置项说明：**

| 配置项 | 说明 |
|--------|------|
| **主配置** | 离线模式、监听端口、统计与成就选项等 |
| **用户配置** | 用户名、语言、存档路径、自定义头像 |
| **DLC 与 Depot** | 解锁全部 DLC 或单独配置 DLC App ID 与名称 |
| **物品与库存** | 配置游戏内物品系统、物品属性与数量 |
| **HTTP 物品** | 配置通过 HTTP 接口获取的物品系统 |
| **成就** | 添加/编辑游戏成就、图标与描述 |
| **统计** | 添加游戏统计数据，支持整数、浮点数、平均速率 |
| **排行榜** | 配置游戏排行榜数据 |
| **模组** | 添加游戏模组并配置加载顺序 |
| **控制器** | 启用控制器支持、摇杆死区、震动强度、按键绑定 |
| **Overlay** | 启用/禁用实验性 Overlay、通知、外观、性能选项 |
| **局域网联机** | 自定义广播地址、自动接受邀请 |
| **DLC 网络** | 配置 DLC 网络相关参数 |
| **应用分组** | 配置应用分组规则 |
| **Depots/Branches** | 配置 Depot 与分支切换规则 |
| **音视频** | 配置音频与视频相关选项 |
| **ColdClient** | 配置 ColdClientLoader 启动参数 |
| **Lobby** | 配置 Lobby Connect 参数 |
| **其他** | 其他高级配置项 |

**保存与同步：**

- 修改任意配置后点击 **保存配置**，配置会写入游戏目录的 `steam_settings` 文件夹
- 保存后会自动广播同步事件，其他已打开的配置窗口会实时刷新
- 单独配置窗口与完整配置管理器共用同一套数据模型，数据始终保持一致

#### 还原文件

若补丁配置后游戏无法正常运行，可点击 **还原文件** 按钮：

1. 程序删除 `steam_settings` 文件夹、补丁注入的 `steam_api.dll` / `steam_api64.dll`、脱壳后的游戏 exe
2. 程序查找所有 `.bak` 备份文件，删除 `.bak` 后缀恢复原文件
3. 还原完成后恢复到补丁应用前的状态

#### Steam 脱壳

某些游戏需要先脱壳才能正常运行：

1. 选择游戏主程序 exe 文件
2. 点击 **Steamless 脱壳** 按钮
3. 程序自动备份原文件并生成脱壳后的 exe

## 五、常见问题

### Q1: 游戏下载进度卡在 0% 不动？

- 检查网络连接是否正常
- 确认清单文件夹路径正确，包含有效的 `.manifest` 文件
- 检查磁盘空间是否充足
- 尝试更换下载路径到其他磁盘
- 检查 `%appdata%/SteamToolPlus/log/{game_id}/` 目录权限

### Q2: 补丁应用后游戏无法启动？

- 检查是否选择了正确的游戏文件夹
- 检查游戏是否放到纯英文路径
- 确保系统杀毒软件没有隔离删除补丁文件的 dll
- 尝试切换标准模式/高级模式
- 尝试切换稳定版/功能版
- 点击 **还原文件** 按钮恢复原始文件后重新配置

### Q3: 如何恢复原始游戏文件？

- 点击补丁页面的 **还原文件** 按钮
- 程序会自动删除补丁文件并恢复 `.bak` 备份

### Q4: 游戏库不显示游戏？

- 检查游戏数据文件 `%appdata%/SteamToolPlus/config/game.json` 是否存在
- 尝试重新导入游戏
- 检查游戏 exe 路径是否正确

### Q5: 背景图片无法显示？

- 确保图片格式为 JPG、PNG 或 WebP
- 检查图片文件是否损坏
- 尝试重新导入背景图片
- 删除 `%appdata%/SteamToolPlus/config/background.json` 后重启程序

### Q6: 程序打开后显示空白/透明窗口？

- 检查是否有杀毒软件拦截了程序
- 尝试删除 `%appdata%/SteamToolPlus/config/` 目录后重新启动程序

### Q7: 如何获取游戏的 Steam App ID？

- 访问 [SteamDB](https://steamdb.info/)
- 搜索游戏名称
- 查看页面上的 App ID
- 或在 Steam 商店页面 URL 中查看数字部分

### Q8: 局域网联机无法找到其他玩家？

- 确保所有玩家在同一局域网内
- 检查防火墙设置，允许程序通过
- 配置正确的广播地址（如 `192.168.1.0/24`）
- 确保所有玩家使用相同的补丁版本

### Q9: 为什么完整配置管理器中看不到 Overlay 配置？

- Overlay 属于功能版（实验版）DLL 的高级功能
- 请先在补丁页面基础配置中勾选“使用实验版（功能版）”，再打开完整配置管理器

### Q10: 首次使用配置弹窗如何再次打开？

- 首次配置弹窗仅在程序首次启动时自动显示
- 如需重新配置，可手动进入【设置】→【全局设置】进行配置

### Q11: 二维码弹窗无法显示？

- 确保网络连接正常
- 尝试刷新页面或重新进入下载页面
- 如仍无法显示，可直接复制网盘链接到浏览器访问

### Q12: 窗口隐藏到托盘后再次打开图片不显示？

- 程序已自动处理：隐藏时释放图片缓存，重新显示时自动重新加载
- 如仍有问题，尝试重启程序
