## 一、软件简介

**Steam Tool Plus** 是一款纯本地运行的 Steam 游戏管理工具，完全绿色便携，无需安装，无需联网，所有数据保存在程序目录下。

### 主要功能

- **游戏库管理**：统一管理本地 Steam 游戏，支持添加、删除、启动、统计游玩时长
- **游戏本体下载**：支持通过 manifest 清单文件下载 Steam 游戏
- **免 Steam 补丁注入**：为游戏注入模拟 Steam 运行环境，无需 Steam 客户端即可启动游戏
- **扩展系统**：支持第三方扩展，可自定义功能

### 系统要求

- Windows 10/11 64位系统
- 无需 Steam 客户端
- 无需联网（除下载游戏外）

## 三、详细使用步骤

### 3.1 游戏库管理

#### 3.1.1 浏览游戏

1. 点击顶部菜单 **游戏** → **浏览** 进入浏览页面
2. 页面顶部显示游戏轮播 Banner
3. 使用筛选栏进行排序和筛选：
   - **排序**：按名称、最近游玩、安装时间、发行时间
   - **筛选**：已安装、未安装、收藏
   - **视图**：网格视图、列表视图

#### 3.1.2 添加游戏

**方式一：手动添加**

1. 点击 **库** 页面右上角的 **+ 添加游戏** 按钮
2. 选择游戏的 .exe 启动文件
3. 程序自动提取游戏名称和图标
4. 可手动修改：
   - 游戏名称
   - 封面图片
   - 启动参数
   - 发行商
   - 发行日期
   - 标签分类
5. 点击 **保存** 完成添加

**方式二：批量扫描**

1. 点击 **批量扫描** 按钮
2. 选择游戏文件夹（包含多个游戏子文件夹）
3. 程序自动扫描目录下的所有 .exe 文件
4. 勾选要添加的游戏
5. 点击 **确认添加**

**方式三：导入 Steam 游戏**

1. 点击 **导入 Steam 游戏** 按钮
2. 程序自动检测本地 Steam 安装目录
3. 列出所有已安装的 Steam 游戏
4. 勾选要导入的游戏
5. 点击 **确认导入**

#### 3.1.3 启动游戏

1. 在游戏卡片上点击 **启动** 按钮
2. 或双击游戏卡片
3. 程序自动记录启动时间
4. 游戏关闭后自动统计游玩时长

#### 3.1.4 管理游戏

- **编辑游戏**：右键游戏卡片 → 编辑，可修改游戏信息
- **删除游戏**：右键游戏卡片 → 删除，仅从库中移除，不删除实际文件
- **收藏游戏**：点击游戏卡片上的星标图标
- **查看详情**：点击游戏卡片进入详情页，查看游玩统计

### 3.2 游戏本体下载

#### 3.2.1 前置准备

1. 从网盘下载游戏的清单文件（manifest）
2. 将下载的清单文件夹放入 `resources/manifest/游戏ID/` 目录
3. 确保文件夹中包含 `.vdf` 和 `.manifest` 文件

#### 3.2.2 单游戏下载

1. 点击顶部菜单 **游戏** → **本体下载**
2. 选择 **单游戏下载** 标签页
3. 点击 **选择清单文件夹** 按钮
4. 选择包含 `.vdf` 和 `.manifest` 文件的文件夹
5. 程序自动验证并提取游戏信息（名称、App ID、清单数量）
6. 选择或确认下载路径（默认 `D:\SteamGame\游戏英文名`）
7. 点击 **开始下载**
8. 查看下载进度和日志输出

#### 3.2.3 批量下载

1. 点击 **批量下载** 标签页
2. 点击 **选择包含清单的文件夹** 按钮
3. 选择包含多个游戏清单文件夹的父文件夹
4. 程序自动扫描并列出所有可下载的游戏
5. 点击 **选择游戏** 按钮
6. 勾选要下载的游戏（支持全选/取消全选）
7. 点击 **确认选择**
8. 选择批量下载的基础路径
9. （可选）勾选 **下载完成后自动关机**
10. 点击 **开始批量下载**

**批量下载特性**：
- 总体进度条显示完成百分比
- 当前下载游戏名称显示
- 每个游戏的单独进度
- 详细的下载日志记录
- 支持随时取消下载

### 3.3 免Steam补丁注入

此功能可为游戏注入模拟 Steam 运行环境，无需 Steam 客户端即可启动游戏。

#### 3.3.1 基础配置

**步骤一：选择游戏文件夹**

1. 点击顶部菜单 **游戏** → **免Steam补丁**
2. 点击 **选择游戏文件夹** 按钮
3. 浏览并选择游戏安装目录
4. 程序自动检测文件夹中的 DLL 文件

**步骤二：选择模拟器模式**

| 模式 | 适用场景 | 检测标志 |
|------|---------|---------|
| **标准模式** | 游戏目录中有 steam_api.dll | 适用于 90% 以上的 Steam 游戏 |
| **高级模式** | 游戏目录中只有 steamclient.dll，或已打 CPY 等破解补丁 | 适用于特殊保护的游戏 |

**步骤三：选择 DLL 版本**（仅标准模式）

| 版本 | 特点 | 推荐度 |
|------|------|--------|
| **稳定版** | 兼容性好，稳定性高，无 Overlay 等高级功能 | ⭐⭐⭐⭐⭐ |
| **功能版** | 支持游戏内 Overlay（Shift+Tab）、成就查看、好友状态等 | ⭐⭐⭐⭐ |

**步骤四：（可选）Steam 脱壳**

某些游戏需要先脱壳才能运行：
1. 点击 **选择游戏主程序** 按钮
2. 选择游戏的 .exe 启动文件
3. 点击 **执行 Steam 脱壳** 按钮
4. 程序自动备份原文件并执行脱壳
5. 脱壳成功后显示绿色勾选标记

**步骤五：输入 Steam AppID**

1. 在输入框中输入游戏的 Steam AppID
2. 可从 Steam 商店页面 URL 获取（如 `store.steampowered.com/app/123456` 中的 123456）
3. 点击 **应用基础配置**
4. 配置成功后显示绿色勾选标记

## 五、注意事项

### 5.1 清单文件获取

游戏下载功能需要先从网盘下载对应的 manifest 文件：
1. 在游戏详情页点击 **清单文件下载** 获取网盘链接
2. 下载后解压到 `resources/manifest/游戏ID/` 目录
3. 确保文件夹中包含 `.vdf` 和 `.manifest` 文件

### 5.2 下载路径

- 默认优先使用 D 盘（`D:\SteamGame\游戏英文名`）
- 如 D 盘不存在，自动回退到 C 盘
- 可在设置中自定义默认下载路径

### 5.3 补丁备份

- 应用补丁前会自动备份原文件
- 备份文件保存在游戏目录的 backup 文件夹中
- 如出现问题可手动从 backup 恢复

### 5.4 脱壳功能

- 某些游戏需要先脱壳才能运行
- 脱壳会自动备份原 exe 文件
- 脱壳后原文件会被替换，备份保存在同目录

### 5.5 DLL 版本选择

- **稳定版**：推荐首选，兼容性最好
- **功能版**：提供更多功能但可能有兼容性问题
- 如游戏运行异常，尝试切换 DLL 版本

### 5.6 Steam AppID 获取

1. 打开 Steam 商店页面
2. 查看 URL 中的数字，如：`store.steampowered.com/app/123456/GameName`
3. AppID 为 `123456`

## 六、常见问题

### Q1: 游戏下载失败怎么办？

- 检查清单文件是否正确放置
- 确认磁盘空间充足
- 检查网络连接
- 查看下载日志获取详细错误信息

### Q2: 补丁应用后游戏无法启动？

- 检查是否选择了正确的游戏文件夹
- 尝试切换 DLL 版本（稳定版/功能版）
- 检查 Steam AppID 是否正确
- 某些游戏需要先执行 Steam 脱壳

### Q3: 如何恢复原始游戏文件？

- 补丁应用时会自动创建 backup 文件夹
- 手动从 backup 文件夹复制原文件覆盖即可

### Q4: 局域网联机无法连接？

- 确保所有玩家使用相同的广播 IP
- 检查防火墙设置
- 确认端口未被占用

### Q5: 扩展加载失败？

- 检查扩展包格式是否正确（必须是 .7z）
- 检查 manifest.json 格式是否正确
- 查看扩展管理页面的错误日志

### Q6: 游戏库不显示游戏？

- 检查游戏数据文件 `data/games.json` 是否存在
- 尝试重新添加游戏
- 检查游戏 exe 路径是否正确
