## 一、工具简介

Steam Tool Plus 是b站up主【鲸落_hi】开发的一款纯本地游戏管理工具，采用简洁现代的UI风格界面设计，支持深色/浅色主题切换。

### 主要功能

| 功能模块 | 说明 |
|---------|------|
| **游戏本体下载** | 使用 DepotDownloader 下载 Steam 游戏文件 |
| **免 Steam 补丁注入** | 通过 Steam 模拟器实现免 Steam 启动游戏 |
| **游戏分类管理** | 支持多种游戏类型的筛选和浏览 |
| **主题切换** | 支持深色/浅色/自动跟随系统主题 |

## 三、详细使用步骤

### 3.1 游戏列表浏览

#### 3.1.1 侧边栏分类导航

点击左侧边栏的不同分类，可筛选显示对应类型的游戏：

- **全部游戏** 🎮 - 显示 games_config.json 中所有游戏
- **免 Steam 启动** ⚡ - tags 中包含 patch_type=0
- **局域网联机** 🌐 - tags 中包含 patch_type=1
- **Steam 联机** 👥 - tags 中包含 patch_type=2
- **D 加密虚拟机** 🛡️ - tags 中包含 patch_type=3

#### 3.1.2 游戏网格布局

- **列数自适应**：内容区域 ≥ 1400px 时显示 **5 列**，否则显示 **4 列**
- **固定间距**：卡片之间始终保持 16px 间距
- **卡片大小**：随窗口拉伸自动调整，始终保持填充满屏幕
- **图片比例**：游戏封面保持原比例填充卡片

#### 3.1.3 游戏卡片交互

| 操作 | 效果 |
|------|------|
| **悬浮** | 卡片放大 1.02 倍，边框高亮显示 |
| **点击** | 进入游戏详情页面 |
| **加载** | 图片懒加载，加载前显示纯色占位符 |

#### 3.1.4 搜索功能

- 在顶部搜索框输入关键词，实时筛选游戏
- 支持搜索：**游戏中文名**、**游戏英文名**、**Game ID**
- 搜索结果即时显示在网格中

#### 3.2.1 游戏下载功能

**前置准备**：
1. 从网盘下载游戏的清单文件（manifest）
2. 点击"清单文件下载"按钮获取网盘链接
3. 将下载的清单文件夹放入 `resources/manifest/游戏ID/` 目录

**下载步骤**：
1. 程序自动检测清单文件夹状态（绿色=已存在，红色=未找到）
2. 选择下载路径（默认自动设置为 `D:\SteamGame\游戏英文名`）
3. 点击"开始下载"按钮
4. 等待下载完成，可查看实时进度和日志

**下载参数说明**：
```
ddv20.exe -lu China --use-http -o "游戏保存文件夹" app -p "清单文件夹"
```

#### 3.2.2 补丁应用功能

**检查补丁状态**：
- 绿色：本地补丁已存在
- 红色：本地补丁未下载

**应用补丁步骤**：

**情况一：本地已有补丁**
1. 点击"浏览"选择游戏安装文件夹
2. 点击"应用补丁"按钮
3. 程序自动备份原文件并应用补丁
4. 查看备份和复制的文件列表确认成功

**情况二：本地没有补丁**
1. 点击"下载补丁"按钮打开网盘链接
2. 下载补丁 .7z 文件到本地
3. 点击"选择补丁文件并应用"
4. 选择下载的 .7z 文件
5. 程序自动解压并应用补丁

**补丁说明**：
- 应用补丁前会自动备份原文件
- 备份文件保存在游戏目录的 backup 文件夹中
- 可点击"查看补丁说明"了解补丁详情

   - App ID
   - 清单文件数量
5. 选择或确认下载路径
6. 点击"开始下载"
7. 查看下载进度和日志输出

#### 3.3.2 批量下载模式

**操作步骤**：
1. 点击"批量下载"标签页
2. 点击"选择包含清单的文件夹"按钮
3. 选择包含多个游戏清单文件夹的父文件夹
4. 程序自动扫描并列出所有可下载的游戏
5. 点击"选择游戏"按钮打开选择弹窗
6. 勾选要下载的游戏（支持全选/取消全选）
7. 点击"确认选择"
8. 选择批量下载的基础路径
9. （可选）勾选"下载完成后自动关机"

**批量下载特性**：
- 总体进度条显示完成百分比
- 当前下载游戏名称显示
- 每个游戏的单独进度
- 详细的下载日志记录

### 3.4 免 Steam 补丁注入页面

点击侧边栏"⚡ 免 Steam 启动"进入补丁注入配置页面。

#### 3.4.1 基础配置

**步骤一：选择游戏文件夹**
1. 点击"选择游戏文件夹"按钮
2. 浏览并选择游戏安装目录
3. 程序自动检测文件夹中的 DLL 文件

**步骤二：选择模拟器模式**

| 模式 | 适用场景 | 检测标志 |
|------|---------|---------|
| **标准模式** | 游戏目录中有 steam_api.dll | 适用于 90% 以上的 Steam 游戏 |
| **高级模式** | 游戏目录中只有 steamclient.dll，或已打 CPY 等破解补丁 | 适用于特殊保护的游戏 |

**步骤三：选择 DLL 版本**（仅标准模式）

| 版本 | 特点 | 推荐度 |
|------|------|--------|
| **稳定版** | 兼容性好，稳定性高，无 Overlay 等高级功能 | ⭐⭐⭐⭐⭐ |
| **功能版** | 支持游戏内 Overlay（Shift+Tab）、成就查看、好友状态等 | ⭐⭐⭐⭐ |

**步骤四：（可选）Steam 脱壳**

某些游戏需要先脱壳才能运行：
1. 点击"选择游戏主程序"按钮
2. 选择游戏的 .exe 启动文件
3. 点击"执行 Steam 脱壳"按钮
4. 程序自动备份原文件并执行脱壳
5. 脱壳成功后显示绿色勾选标记

**步骤五：输入 Steam AppID**
1. 在输入框中输入游戏的 Steam AppID
2. 可从 Steam 商店页面 URL 获取（如 `store.steampowered.com/app/123456` 中的 123456）
3. 点击"应用基础配置"
4. 配置成功后显示绿色勾选标记

## 五、注意事项

### 5.1 清单文件获取

游戏下载功能需要先从网盘下载对应的 manifest 文件：
1. 在游戏详情页点击"清单文件下载"获取网盘链接
2. 下载后解压到 `resources/manifest/游戏ID/` 目录
3. 确保文件夹中包含 `.vdf` 和 `.manifest` 文件

### 5.2 下载路径

- 默认优先使用 D 盘（`D:\SteamGame\游戏英文名`）
- 如 D 盘不存在，自动回退到 C 盘
- 可在设置中自定义默认下载路径

### 5.3 补丁备份

- 应用补丁前会自动备份原文件
- 备份文件保存在游戏目录的 backup 文件夹中
- 如出现问题可手动从 backup 恢复

### 5.4 脱壳功能

- 某些游戏需要先脱壳才能运行
- 脱壳会自动备份原 exe 文件
- 脱壳后原文件会被替换，备份保存在同目录

### 5.5 DLL 版本选择

- **稳定版**：推荐首选，兼容性最好
- **功能版**：提供更多功能但可能有兼容性问题
- 如游戏运行异常，尝试切换 DLL 版本

### 5.6 Steam AppID 获取

1. 打开 Steam 商店页面
2. 查看 URL 中的数字，如：`store.steampowered.com/app/123456/GameName`
3. AppID 为 `123456`

## 八、常见问题

### Q1: 游戏下载失败怎么办？
- 检查清单文件是否正确放置
- 确认磁盘空间充足
- 检查网络连接
- 查看下载日志获取详细错误信息

### Q2: 补丁应用后游戏无法启动？
- 检查是否选择了正确的游戏文件夹
- 尝试切换 DLL 版本（稳定版/功能版）
- 检查 Steam AppID 是否正确
- 某些游戏需要先执行 Steam 脱壳

### Q3: 如何恢复原始游戏文件？
- 补丁应用时会自动创建 backup 文件夹
- 手动从 backup 文件夹复制原文件覆盖即可

### Q4: 局域网联机无法连接？
- 确保所有玩家使用相同的广播 IP
- 检查防火墙设置
- 确认端口未被占用
