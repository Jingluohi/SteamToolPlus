## 一、软件简介

**Steam Tool Plus** 是一款纯本地运行的 Steam 游戏管理工具，完全绿色便携，无需安装，无需联网，所有数据保存在程序目录下。

### 主要功能

- **游戏库管理**：统一管理本地 Steam 游戏，支持添加、删除、启动、统计游玩时长
- **游戏本体下载**：支持通过 manifest 清单文件下载 Steam 游戏
- **免 Steam 补丁注入**：为游戏注入模拟 Steam 运行环境，无需 Steam 客户端即可启动游戏
- **清单入库**：将清单文件导入 Steam，使游戏库中显示对应游戏
- **扩展系统**：支持第三方扩展，可自定义功能

### 系统要求

- Windows 10/11 64位系统
- 无需 Steam 客户端
- 无需联网（除下载游戏外）

## 三、详细使用步骤

### 3.1 游戏库管理

#### 3.1.1 浏览游戏

1. 点击顶部菜单 **游戏** → **浏览** 进入浏览页面
2. 页面显示所有可下载的游戏
3. 使用搜索框搜索游戏
4. 点击游戏卡片进入详情页

#### 3.1.2 游戏库

1. 点击顶部菜单 **游戏** → **库** 进入游戏库页面
2. 显示所有已下载或导入的游戏
3. 支持启动游戏、编辑游戏信息、卸载游戏

**添加游戏到库：**

- 下载游戏后自动添加到库
- 手动导入：点击 **导入游戏** 按钮，选择游戏目录和主程序 exe

**启动游戏：**

1. 在游戏卡片上点击 **开始游戏** 按钮
2. 程序自动记录启动时间和游玩时长
3. 游戏运行时按钮变为 **关闭游戏**，可结束游戏进程

**管理游戏：**

- **编辑游戏**：点击编辑按钮，可修改游戏目录、exe 路径、存档路径、封面
- **卸载游戏**：右键游戏卡片 → 卸载，删除游戏文件但保留游戏数据
- **浏览本地文件**：右键游戏卡片 → 浏览本地文件
- **收藏游戏**：右键游戏卡片 → 收藏/取消收藏

### 3.2 游戏本体下载

#### 3.2.1 前置准备

1. 从网盘下载游戏的清单文件（manifest）
2. 将下载的清单文件夹放入程序目录
3. 确保文件夹中包含 `.vdf` 和 `.manifest` 文件

#### 3.2.2 单游戏下载

1. 点击顶部菜单 **工具** → **本体下载**
2. 点击 **选择清单文件夹** 按钮
3. 选择包含 `.vdf` 和 `.manifest` 文件的文件夹
4. 程序自动验证并提取游戏信息（名称、App ID、清单数量）
5. 选择或确认下载路径
6. 点击 **开始下载**
7. 查看下载进度和日志输出

**下载控制：**

- **暂停下载**：点击暂停按钮可停止 ddv20.exe 进程
- **恢复下载**：重新进入详情页会自动恢复下载状态
- **下载完成**：自动清理 100% 开头的 JSON 进度文件

### 3.3 免Steam补丁注入

此功能可为游戏注入模拟 Steam 运行环境，无需 Steam 客户端即可启动游戏。

#### 3.3.1 基础配置

**步骤一：选择游戏文件夹**

1. 点击顶部菜单 **工具** → **免Steam补丁**
2. 点击 **选择游戏文件夹** 按钮
3. 浏览并选择游戏安装目录
4. 程序自动检测文件夹中的 DLL 文件

**步骤二：选择模拟器模式**

| 模式 | 说明 |
|------|------|
| **标准模式** | 替换 steam_api.dll，适用于 90% 以上的 Steam 游戏 |
| **高级模式** | 替换 steamclient.dll + steam_api.dll，适用于特殊游戏 |

**步骤三：选择 DLL 版本（标准模式下）**

| 版本 | 说明 |
|------|------|
| **稳定版** | 推荐大多数用户使用，兼容性好，稳定性高 |
| **功能版** | 包含 Overlay 等高级功能，部分游戏可能有兼容性问题 |

**步骤四：配置 Steam App ID**

1. 在 **Steam App ID** 输入框中填写游戏的 App ID
2. 点击 **SteamDB** 链接可查询游戏 App ID
3. 程序会自动生成 steam_interfaces.txt 文件

**步骤五：应用配置**

1. 点击 **应用配置** 按钮
2. 程序自动复制补丁文件到游戏目录
3. 可直接从游戏库启动游戏

## 五、注意事项

### 5.1 清单文件获取

游戏下载功能需要先从网盘下载对应的 manifest 文件：
1. 在游戏详情页查看下载链接
2. 下载后解压到程序目录
3. 确保文件夹中包含 `.vdf` 和 `.manifest` 文件

### 5.2 下载路径

- 默认优先使用 D 盘
- 可在设置中自定义默认下载路径

### 5.3 补丁备份

- 应用补丁前会自动备份原文件
- 备份文件保存在游戏目录的 backup 文件夹中
- 如出现问题可手动从 backup 恢复

### 5.4 Steam AppID 获取

1. 打开 Steam 商店页面
2. 查看 URL 中的数字，如：`store.steampowered.com/app/123456/GameName`
3. AppID 为 `123456`

### 5.5 配置文件位置

- **运行时配置**：`%appdata%/SteamToolPlus/config/`
- **备份配置**：程序根目录 `config/`
- **游戏数据**：`%appdata%/SteamToolPlus/config/game.json`

## 六、常见问题

### Q1: 游戏下载进度卡在 0% 不动？

- 检查网络连接是否正常
- 确认清单文件夹路径正确，包含有效的 manifest 文件
- 检查磁盘空间是否充足
- 尝试更换下载路径到其他磁盘

### Q2: 补丁应用后游戏无法启动？

- 检查是否选择了正确的游戏文件夹
- 检查游戏是否放到纯英文路径
- 确保系统杀毒软件没有隔离删除补丁文件的 dll
- 尝试切换标准模式/高级模式
- 尝试切换稳定版/功能版

### Q3: 如何恢复原始游戏文件？

- 补丁应用时会自动创建 backup 文件夹
- 手动从 backup 文件夹复制原文件覆盖即可

### Q4: 游戏库不显示游戏？

- 检查游戏数据文件 `%appdata%/SteamToolPlus/config/game.json` 是否存在
- 尝试重新导入游戏
- 检查游戏 exe 路径是否正确

### Q5: 背景图片无法显示？

- 确保图片格式为 JPG、PNG 或 WebP
- 检查图片文件是否损坏
- 尝试重新导入背景图片
- 删除 `%appdata%/SteamToolPlus/config/background.json` 后重启程序

### Q6: 程序打开后显示空白/透明窗口？

- 确保程序根目录下存在 `resources/pic/background/` 背景图片
- 检查是否有杀毒软件拦截了程序
- 尝试删除 `%appdata%/SteamToolPlus/config/` 目录后重新启动程序

### Q7: 如何获取游戏的 Steam App ID？

- 访问 [SteamDB](https://steamdb.info/)
- 搜索游戏名称
- 查看页面上的 App ID
- 或在 Steam 商店页面 URL 中查看数字部分

### Q8: 局域网联机无法找到其他玩家？

- 确保所有玩家在同一局域网内
- 检查防火墙设置，允许程序通过
- 配置正确的广播地址（如 `192.168.1.0/24`）
- 确保所有玩家使用相同的补丁版本
